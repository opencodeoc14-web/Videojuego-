# Turbo Legends — prototipo jugable de 8 jugadores

Prototipo web 3D funcional de carreras arcade.

## Jugadores integrados

- Astra — Fénix X
- Mara Volt — Cometa Rosa
- Glacior — Rompehielo
- Nébulo — Espectra
- Profesor Bujía — Chispa 7
- Riff Colmillo — Garra ATV
- Gorak — Martillo
- Barón Vórtice — Vórtice Real

Cada jugador posee representación 3D procedural, colores, silueta, estadísticas y habilidad propia.

## Jugabilidad actual

- Carrera temporal de prueba contra siete rivales controlados por IA.
- Tres vueltas, clasificación en vivo y podio.
- Dirección, derrape, mini-turbo, plataformas turbo, salto, colisiones y pausa.
- Controles táctiles para móvil y teclado para escritorio.
- Diseño adaptable.

Las cinco pistas definitivas aparecen bloqueadas y se construirán en la siguiente fase.

## Tecnología

JavaScript modular, Three.js y WebGL. GitHub Actions publica los archivos estáticos en GitHub Pages.
